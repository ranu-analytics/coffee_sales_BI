## Coffee Sales Dashboard 

![Coffee Dashboard](coffee-dashboard.png)

---

###  Project Overview  
This project showcases an interactive Coffee Sales Dashboard built using Power BI. It provides insights into sales trends, product performance, and customer purchasing behavior.

---

###  Key Metrics  
-  Total Revenue: 115.43K  
-  Sales Trends by Month  
-  Sales by Time of Day  
-  Product-wise Sales Analysis  

---

###  Insights  
- Peak sales observed during afternoon hours  
- Latte and Americano are top-selling products  
- Sales trend shows gradual decline over months  
- Morning and afternoon contribute highest revenue  

---

###  Key Highlights  
- Strong demand for premium coffee products  
- Consistent customer purchasing patterns  
- Clear time-based sales behavior  

---

###  Tools Used  
- Power BI  
- DAX  
- Data Visualization
